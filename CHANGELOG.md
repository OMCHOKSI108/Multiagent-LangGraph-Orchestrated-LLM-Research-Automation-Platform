# Changelog

All notable changes to the Multi-Agent LLM Research Automation Platform will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of the Multi-Agent LLM Research Automation Platform
- Multi-agent architecture with 15+ specialized agents
- Support for Ollama, Google Gemini, and Groq LLM providers
- Two main pipelines: Full Research Synthesis and Single Paper Analysis
- React frontend with modern UI components
- Node.js backend with authentication and API management
- FastAPI-based AI Engine with LangGraph orchestration
- Comprehensive documentation and setup guides
- Docker containerization for all services
- GitHub Actions CI/CD pipeline
- Token tracking and cost optimization
- Real-time progress monitoring

### Technical Features
- LangGraph-powered agent orchestration
- Multi-provider LLM abstraction layer
- PDF processing with PyMuPDF and Unstructured
- PostgreSQL database with connection pooling
- JWT-based authentication
- Rate limiting and security middleware
- Comprehensive logging and error handling
- Test suites for all components
