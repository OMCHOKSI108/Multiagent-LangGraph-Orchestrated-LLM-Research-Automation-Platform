# Deep Research Engine

![Banner](../assets/banner.png)

## Overview

The **Deep Research Engine** is an advanced, autonomous multi-agent platform designed to conduct comprehensive research on any topic. Orchestrated by **LangGraph**, it coordinates specialized AI agents to scrape, analyze, critique, and synthesize information into professional-grade reports.

## Key Features

- **Autonomous Research**: Simply provide a topic, and the engine handles the entire research lifecycle.
- **Multi-Agent Architecture**: Specialized agents for scraping, critical review, and report generation working in harmony.
- **Verified Sources**: Aggregates data from Google, Wikipedia, Arxiv, PubMed, and more.
- **Professional Reports**: Generates detailed reports in Markdown, PDF, and LaTeX formats.
- **Real-Time Visualization**: Watch the agents "think" and execute tasks live via the interactive dashboard.

## Quick Start

### Installation

```bash
git clone https://github.com/OMCHOKSI108/Multiagent-LangGraph-Orchestrated-LLM-Research-Automation-Platform.git
cd Multiagent-LangGraph-Orchestrated-LLM-Research-Automation-Platform
python run.py
```

### Usage

1. Open the dashboard at `http://localhost:3000`.
2. Enter a research topic (e.g., "The Future of Quantum Computing").
3. Configure depth and breadth settings.
4. Click **Start Research**.
